# 域名配置与网站更新指南

## 1. 域名配置 (在阿里云控制台操作)

由于我无法登录您的阿里云账号，请您**复制以下信息**，填入阿里云“域名解析”页面：

### 第一条记录 (让您的域名 qixiaoyu.com 能访问)
*   **记录类型**: `A`
*   **主机记录**: `@`
*   **记录值**: `76.76.21.21`

### 第二条记录 (让 www.qixiaoyu.com 能访问)
*   **记录类型**: `CNAME`
*   **主机记录**: `www`
*   **记录值**: `cname.vercel-dns.com`

---

## 2. 网站部署 (在 Vercel 操作)

1.  登录 [Vercel.com](https://vercel.com)
2.  点击 **Add New...** -> **Project**
3.  选择 **Import** 您的 GitHub 仓库: `personal-portfolio`
4.  点击 **Deploy** (部署)
5.  部署完成后，点击顶部 **Settings** -> **Domains**
6.  输入您的域名 `qixiaoyu.com` 并点击 **Add**

---

## 3. 以后如何更新网站？

您只需要在本地做两件事，网站就会自动更新：

1.  **修改代码/上传图片**
2.  **点击“同步”** (或者在对话框里告诉我“同步到GitHub”)

一旦同步成功，Vercel 会自动检测到变化，并自动为您发布新版本（通常需要1-2分钟）。
